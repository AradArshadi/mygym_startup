# myGym v0.9.3.4 — Clarity UX Hotfix

This hotfix keeps the Fitness Home direction but simplifies the navigation so the current product is easier to understand.

## Changed

- Replaced customer mobile bottom nav with: Home / Explore / + / MyGym / More.
- Moved unfinished Discover and Chat features out of the primary nav into the More page.
- Renamed Profile hub to MyGym to better match sessions, access passes, workouts, saved gyms, and reviews.
- Added clearer Home quick actions: Explore gyms, Access pass, Log workout, MyGym.
- Added a More page for notifications, goal settings, workout history, theme toggle, logout, and future feature previews.
- Fixed the malformed Fitness Home title block from the previous hotfix package.

## No migration required

This is a template/CSS/navigation hotfix only.
